INPUT TEKS: 
1. Jalankan vs code Buka file (KRIPTOGRAFI_BURHAN.py) 
2. Jalankan program 
3. Masukkan input berupa teks pada samping "Enter teks:" 
4. Masukkan key 
5. Pilih chiper antara Vigenere Cipher, Playfair Cipher, atau Hill Cipher 
6. Pilih enkripsi/dekripsi 
7. Tekan Action 
8. Hasil enkripsi/dekripsi akan keluar

INPUT FILE (.txt): 
1. Jalankan vs code 
2. Tekan tombol "Load File" 
3. Pilih file (.txt) yang akan di enkripsi/dekripsi 
4. Masukkan key 
5. Pilih chiper antara Vigenere Cipher, Playfair Cipher, atau Hill Cipher 
6. Pilih enkripsi/dekripsi 
7. Tekan tombol "Action" 
8. Hasil enkripsi/dekripsi akan keluar
